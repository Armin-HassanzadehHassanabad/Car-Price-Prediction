Car Price Prediction

This project builds a machine learning model to predict car prices using a dataset from Kaggle.

Overview
The workflow includes:
- Data cleaning and preprocessing
- Feature engineering
- Exploratory Data Analysis (EDA)
- Encoding categorical variables
- Feature selection using RFE and VIF
- Linear regression modeling with statistical analysis
- Model evaluation using R² score and residual analysis

Results
The final model achieves strong predictive performance with good alignment between actual and predicted prices.

Tools & Libraries
- Python
- pandas, numpy
- matplotlib, seaborn
- scikit-learn
- statsmodels

Notes
This project is part of my machine learning practice and focuses on understanding the regression pipeline.
